# VPS Info & Logs

Script para visualizar informações e logs de um servidor VPS.

## Execução

O script deve ser executado como root.

```
git clone https://github.com/ricjcs/vps-info-logs
cd vps-info-logs
sudo bash vps-info-logs.sh
```
